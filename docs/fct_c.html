Concatenate factors, unioning levels.

Usage

fct_c(fs)

Arguments

fs
A list of factors

Description

This is useful way of patching together factors from multiple sources that really should have the same levels but don't.

Examples

fs <- list(factor("a"), factor("b"), factor(c("a", "b")))
fct_c(fs)


[1] a b a b
Levels: a b
